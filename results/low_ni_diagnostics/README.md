# Low-Ni WN3 Shock-Cooling Diagnostics

This directory backs up the low-nickel WN3 shock-cooling plots and CSV summaries generated during the SNEC early-time analysis.

The matching plotting and analysis scripts are stored in:

`scripts/low_ni_diagnostics/`

Key outputs include:

- `low_ni_paper_variable_treatment_comparison.png`: SNEC g-band curves, Planck recovery, and compact Nakar/Sari variable-ratio diagnostics.
- `low_ni_gband_luminosity_tau_root_comparison.png`: g-band luminosity comparison using the tau=2/3 photosphere root.
- `low_ni_tau_root_photosphere_turnover.png`: photospheric radius root-finding check against SNEC `rad_photo.dat`.
- `low_ni_log_density_vs_log_radius.png`: density structure versus radius.
- `low_ni_wn3_composition_materials.png`: WN3 composition/material structure.

Copied 16 scripts and 46 result files.
